## 🎨 Features

- **Clean, Modern Design**: Soft color palette with professional aesthetics
- **Responsive Layout**: Fully optimized for desktop, tablet, and mobile devices
- **Modular Architecture**: Separated HTML, CSS, and JavaScript for easy maintenance
- **Smooth Animations**: Subtle transitions and animations for enhanced UX
- **Easy Navigation**: Intuitive sidebar navigation with smooth section switching
- **Contact Form**: Fully functional contact form with validation
- **Accessible**: Semantic HTML and ARIA labels for accessibility
- **Performance Optimized**: Lightweight and fast-loading

## 🚀 Quick Start

### Installation

1. Clone or download the repository
2. Place your profile image in the `assets/` folder as `profile-image.jpg`
3. Update content as needed in `index.html`
4. Ensure all files are in the correct directory structure
